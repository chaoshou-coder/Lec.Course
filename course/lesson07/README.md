# Day 07 · 代码结构与多页面网站整合

> 今日节点: n12(代码结构与注释) + n13(多页面网站整合)
> 预计时长: 2 小时
> 今日目标: 写出可读性强的代码,并整合多页面网站

## 今日知识点

1. **代码可读性** — 缩进、嵌套、空行、一致的代码风格
2. **HTML 注释** — `<!-- -->` 的用法:说明、标记、临时隐藏
3. **多页面链接** — a 标签在同一站点内多个页面间导航
4. **项目目录结构** — HTML 文件 + 图片/CSS 的组织方式
