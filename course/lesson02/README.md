# Day 02 · 文本元素:让文章有层级和节奏

> 今日节点: n3(文本元素 h1-h6/p/br/hr)
> 预计时长: 1.5 小时
> 今日目标: 用 h1-h6/p/br/hr 写出带层级的文章内容

## 今日知识点

1. **标题标签 h1-h6** — 6 级标题,层级递减,一个页面通常只有一个 h1
2. **段落标签 p** — 段落容器,浏览器自动在段落间加间距
3. **换行 br 与分隔线 hr** — 段落内换行(br)和主题间分隔(hr)

## 学习路径

```
看笔记 notes.md → 完成 6 道当堂练 → 挑战 3 道课后作业
```
