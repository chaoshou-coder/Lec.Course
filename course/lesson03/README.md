# Day 03 · 链接、图片与列表:让页面"活"起来

> 今日节点: n4(超链接 a 标签/href/target) + n5(图片 img/alt/src) + n6(列表 ul/ol/li)
> 预计时长: 2 小时
> 今日目标: 插入链接和图片,用列表组织内容

## 今日知识点

1. **超链接 a 标签** — href 指定跳转地址,target 控制打开方式(_blank 新标签页)
2. **图片 img 标签** — src 指定图片路径,alt 提供替代文本(必填)
3. **列表 ul/ol/li** — ul 无序列表(圆点),ol 有序列表(数字),li 列表项
