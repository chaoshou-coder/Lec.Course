## 今日知识点

1. **table 标签** — 表格容器,所有行和单元格都放在里面
2. **tr 标签** — 表格行(table row),每行用一个 tr 包裹
3. **td 标签** — 数据单元格(table data),存放具体数据
4. **th 标签** — 表头单元格(table header),加粗居中,标记列含义
