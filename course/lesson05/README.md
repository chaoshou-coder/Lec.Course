# Day 05 · 表单基础:让页面"接收"用户输入

> 今日节点: n8(表单基础 form/action/method) + n9(表单控件 input/select/textarea)
> 预计时长: 2 小时
> 今日目标: 用 form + input/select/textarea 制作用户输入界面

## 今日知识点

1. **表单 form** — 表单容器,action 指定提交目标,method 指定提交方式(GET/POST)
2. **input 控件** — 文本框/密码框/邮箱/数字/单选/复选框等输入类型
3. **select 下拉框** — 从多个选项中选择一个
4. **textarea 多行文本** — 接收大段文字输入
