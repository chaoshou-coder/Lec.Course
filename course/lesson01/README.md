# Day 01 · 你的第一个 HTML 文档

> 今日节点: n1(HTML 标签与属性语法) + n2(文档基本结构)
> 预计时长: 2 小时
> 今日目标: 写出第一个可渲染的 HTML 文档

## 今日知识点

1. **HTML 标签与属性语法** — 什么是标签、开标签/闭标签、自闭合标签、属性
2. **文档基本结构** — `<!DOCTYPE html>` / `<html>` / `<head>` / `<body>` / `<title>`

## 学习路径

```
看笔记 notes.md → 完成 6 道当堂练 → 挑战 3 道课后作业
```
