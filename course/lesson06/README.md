# Day 06 · 表单进阶与语义化标签

> 今日节点: n10(表单提交 button/label/字段分组) + n11(语义化标签 header/nav/main/section/article/footer)
> 预计时长: 2 小时
> 今日目标: 用 label/fieldset + 语义化标签制作专业级页面结构

## 今日知识点

1. **button 标签** — 提交按钮,type="submit/button/reset" 三种形态
2. **label 标签** — 关联输入控件,提升可访问性和点击体验
3. **fieldset/legend** — 表单字段分组,视觉和语义上的归类
4. **语义化标签** — header/nav/main/section/article/footer 替代 div
