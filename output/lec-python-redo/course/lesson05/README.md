# Day 05 · 函数

> 今日节点: d05(def/参数/返回值/作用域/模块化)
> 预计时长: 6 小时
> 教学法: NCDL(先感知场景再抽象规则)

## 今日知识点

- def 定义/调用
- 函数四种形式(无参无返/有参无返/无参有返/有参有返)
- 形参 vs 实参
- 默认参数(必须放最后)
- 关键字参数
- return(带结果回家+立刻退出)
- 多值返回(元组解包)
- 作用域(全局 vs 局部)
- 函数封装与模块化
