## 今日知识点

- class 定义类(大驼峰命名)
- __init__ 构造函数(self 代表当前对象)
- 实例方法(第一个参数 self)
- 实例属性 vs 类属性
- @property(getter/setter 校验)
- __str__ 与 __repr__
- 继承(class 子类(父类))
- super() 调用父类方法
- 业务叙事锚点:电商订单系统贯穿 4 天
