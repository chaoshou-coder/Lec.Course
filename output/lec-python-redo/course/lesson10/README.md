## 今日知识点

- 需求分析与功能清单
- 数据结构设计(list[dict])
- 函数拆分原则(每函数 ≤ 40 行/只做一件事)
- while True 菜单循环
- try-except 异常加固
- JSON 文件持久化(load/dump)
- 项目架构分层(数据层/功能层/持久层)
- 购物车系统:商品库 → 加购 → 查看 → 结算 → 持久化
