### Day 01 · Python 入门:你的第一个程序

> **痛点**:你看过无数"学 Python"的帖子,但从未想过网页、App、AI 背后都是代码在运行。今天你将写出人生第一个 Python 程序 —— 在屏幕上打印"你好,世界",并让电脑问你叫什么名字。
> **类比**:Python 解释器就像"翻译官" —— 你用中文(代码)下指令,翻译官转成电脑能执行的机器语言。
> **解释**:**程序 = 给计算机的指令清单**。今天学:输出、输入、变量、注释。

---

#### 输出 —— 让电脑说话

> **痛点**:你打开 Python 却不知道让它干什么。第一个能力:让电脑显示文字。
> **类比**:`print()` 就像"让电脑开口说话" —— 你说什么,它显示什么。
> **解释**:`print()` 是 Python 的输出函数,把括号里的内容显示到屏幕。

```python
# 你的第一行代码(在 VS Code 新建 hello.py,输入下面代码,按 F5 运行)

print("你好,世界!")
print("我是 Python,今天开始学编程")

# 输出多个内容(用逗号分隔,自动加空格)
print("我今年", 25, "岁")

# 输出空行
print()
print("上面是空行")
```

**逐行解剖**

- `print("你好,世界!")` = 把字符串显示到屏幕
- `print("我今年", 25, "岁")` = 输出多个参数,自动加空格 → `我今年 25 岁`
- `print()` = 输出空行

> **ASCII 执行过程**
> ```
> print("你好,世界!")
>    │
>    ▼
> [Python 解释器]
>    │
>    ▼
> 屏幕显示:你好,世界!
> ```

**常见错误**

> 1. **错误现象**:`SyntaxError: invalid syntax`(忘记引号)
>    **原因:**字符串必须用引号包裹。修正:`print("你好")` 不是 `print(你好)`
> 2. **错误现象**:`NameError: name 'xxx' is not defined`(变量未定义)
>    **原因:**没赋值就用。修正:先赋值 `name = "Alice"` 再用

#### 输入 —— 让电脑问你

> **痛点**:程序每次运行都一样,你想让它根据你的输入做不同的事。
> **类比**:`input()` 就像"让电脑问你问题" —— 你输入答案,程序接着执行。
> **解释**:`input()` 暂停程序,等用户输入,返回输入的字符串。

```python
# 让电脑问你名字
name = input("请输入你的名字:")
print("你好,", name, "!")

# input() 返回字符串,算术运算前要转数字
age = input("请输入你的年龄:")
age = int(age)  # 字符串转整数
print("明年你", age + 1, "岁")
```

**逐行解剖**

- `input("请输入你的名字:")` = 显示提示,等用户输入,返回字符串
- `name = input(...)` = 把输入赋值给变量 name
- `int(age)` = 把字符串转整数(否则 `age + 1` 会报错或拼接)

**常见错误**

> 1. **错误现象**:`TypeError: can only concatenate str to str`(字符串 + 数字)
>    **原因:**`input()` 返回字符串,不能直接和数字运算。修正:`int(input(...))`
> 2. **错误现象**:输入字母转整数时报 `ValueError`
>    **原因:**`int("abc")` 不合法。修正:先判断 `isdigit()` 再转

#### 变量 —— 数据的"贴标签盒子"

> **痛点**:你拿到数据(用户输入/计算结果)后,需要存起来以后用。
> **类比**:变量就像"贴了标签的盒子" —— 你把数据放进去,以后用标签名取。
> **解释**:**变量 = 名字 + 类型 + 值**。用 `=` 赋值。

```python
# 变量赋值
name = "Alice"       # 字符串
age = 25             # 整数
height = 1.68        # 浮点数
is_student = True    # 布尔

# 查看变量类型
print(type(name))     # <class 'str'>
print(type(age))      # <class 'int'>

# 查看变量地址(唯一标识)
print(id(name))       # 140234567890

# 变量重新赋值
age = 26             # 现在 age 是 26
print("明年", age, "岁")
```

**逐行解剖**

- `name = "Alice"` = 创建变量 name,赋值 "Alice"
- `type(name)` = 查看变量类型
- `id(name)` = 查看变量在内存中的地址
- `age = 26` = 重新赋值(覆盖旧值)

> **ASCII 变量模型**
> ```
> 变量表(内存)
> ┌──────┬──────────────┐
> │ name │ "Alice"      │
> │ age  │ 26           │
> │height│ 1.68         │
> └──────┴──────────────┘
> ```

**常见错误**

> 1. **错误现象**:变量名写错大小写
>    **原因:**Python 区分大小写。`name` 和 `Name` 是不同变量
> 2. **错误现象**:用未赋值的变量
>    **原因:**变量必须先赋值再用。修正:先 `name = "Alice"` 再 `print(name)`

#### 注释 —— 给代码写"说明书"

> **痛点**:你一个月前写的代码,现在看不懂。
> **类比**:注释就像"给代码贴便签" —— 解释这段代码在干什么。
> **解释**:注释是给程序员看的,Python 解释器会忽略。

```python
# 这是单行注释(井号后面的内容被忽略)

"""
这是多行注释(三引号包裹)
可以写多行
常用于函数/类的文档说明
"""

name = "Alice"  # 行内注释(代码后面的注释)
```

> 2. **错误现象**:`TypeError: can only concatenate str to str`
>    **原因:**字符串不能直接 + 数字。修正:`str(age)` 或 f-string
