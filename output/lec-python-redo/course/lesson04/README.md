# Day 04 · 循环入门

> 今日节点: d04(while/for/range/break/continue/嵌套循环)
> 预计时长: 6 小时
> 教学法: NCDL(先感知场景再抽象规则)

## 今日知识点

- while 循环(起始值/终止条件/递增步进/死循环风险)
- break 与 continue
- for item in 序列(遍历字符串/列表)
- range(start, stop, step)
- 累加器与累乘器
- end='' 参数控制输出不换行
- 嵌套循环(九九乘法表/图形打印)
- 鸡兔同笼(循环+条件)
- 菜单循环(while True + break)
- 字符串重复(* 运算符)
