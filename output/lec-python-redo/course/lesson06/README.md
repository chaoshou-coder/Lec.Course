# Day 06 · 列表与字典

> 今日节点: d06(列表/字典/索引/切片/CRUD/排序/遍历)
> 预计时长: 6 小时
> 教学法: NCDL(先感知场景再抽象规则)

## 今日知识点

- 列表创建(有序/可变/可重复)
- 列表索引与切片(同字符串,支持逆序 [::-1])
- 列表可变性(原地修改 vs 字符串不可变)
- 列表 CRUD(append/insert/extend/pop/remove/clear)
- 列表排序与查询(sort/index/count/in)
- 字典创建(key-value 对)
- 字典取值(d[key] vs d.get(key, 默认值))
- 字典 CRUD(增/改/删 pop)
- 字典遍历(keys()/values()/items())
