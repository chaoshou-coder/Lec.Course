# Day 03 · 条件分支

> 今日节点: d03(比较运算/if/elif/else/逻辑运算/三元表达式)
> 预计时长: 6 小时
> 教学法: NCDL(先感知场景再抽象规则)

## 今日知识点

- 比较运算符(> < == != >= <=, = vs ==)
- if 语句(冒号+缩进 4 格)
- if-else / if-elif-else(多选一)
- 逻辑运算(and or not)
- 嵌套条件(不超过两层)
- 三元表达式(值1 if 条件 else 值2)
- in 运算符
- 综合实战(BMI 计算器/闰年判断)
