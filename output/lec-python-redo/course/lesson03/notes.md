### Day 03 · 条件分支:让程序"会思考"

> **痛点**:你的程序只会"顺序执行",不能根据不同情况做不同的事。今天你将让程序拥有"判断力" —— 像红绿灯一样,红灯停、绿灯行。
> **类比**:条件分支就像"岔路口" —— 程序走到路口,根据条件选左还是选右。
> **解释**:**条件分支 = 根据条件决定执行哪段代码**。今天学:if/elif/else、逻辑运算、三元表达式。

---

#### 比较运算符 —— 判断的"尺子"

> **痛点**:你想判断两个数谁大、两个字符串是否相等,不知道怎么写。
> **类比**:比较运算符就像"尺子" —— 量一量谁大谁小,结果是 True 或 False。
> **解释**:比较运算符返回布尔值(True/False),最常用的是 `==`(等于)和 `=`(赋值)的区别。

```python
# 比较运算符
print(5 > 3)     # True(大于)
print(5 < 3)     # False(小于)
print(5 == 5)    # True(等于,注意是两个等号)
print(5 != 3)    # True(不等于)
print(5 >= 5)    # True(大于等于)
print(5 <= 3)    # False(小于等于)

# = 赋值 vs == 比较(最常见的错误!)
x = 5            # 赋值:把 5 放进 x
print(x == 5)    # 比较:x 是否等于 5? → True
```

**逐行解剖**

- `>` `<` `>=` `<=` = 大小比较
- `==` = 判断相等(两个等号,问"相等吗?")
- `!=` = 判断不相等
- `=` = 赋值(一个等号,把右边放进左边)

> **ASCII 比较示意图**
> ```
> 比较运算符返回布尔值:
>
> 5 == 5  →  True  (相等吗? 是的)
> 5 != 5  →  False (不相等吗? 不,相等)
> 5 > 3   →  True  (大于吗? 是的)
>
> = vs ==:
> x = 5       → 把 5 赋值给 x
> x == 5      → 问:x 等于 5 吗?
> ```

**常见错误**

> 1. **错误现象**:`if x = 5:` 报错
>    **原因:**`=` 是赋值,`==` 才是比较。修正:`if x == 5:`
> 2. **错误现象**:混淆 `==` 和 `is`
>    **原因:**`==` 比较值,`is` 比较内存地址。判断值相等用 `==`

#### if 语句 —— 最基本的判断

> **痛点**:你想让程序"如果年龄大于等于 18 就输出成年",不知道怎么写。
> **类比**:if 就像"如果...就..." —— 条件成立就执行,不成立就跳过。
> **解释**:`if 条件:` 后面跟冒号和 4 格缩进,缩进的代码是条件成立时执行的。

```python
age = 20

# 基本 if 语句
if age >= 18:
    print("你已成年")        # 缩进 4 格,条件成立才执行
    print("可以考驾照")      # 同一缩进,一起执行

print("程序结束")            # 没缩进,无论条件如何都执行

# 条件不成立,跳过
age = 15
if age >= 18:
    print("你已成年")        # 不执行(条件为 False)
print("程序结束")
```

**逐行解剖**

- `if age >= 18:` = 判断条件,末尾必须有冒号
- 缩进 4 格 = Python 用缩进表示代码块(不是大括号!)
- 没缩进的代码 = 不属于 if,始终执行

> **ASCII 执行流程图**
> ```
> age = 20
>    │
>    ▼
> age >= 18 ?
>    │
> ┌──┴──┐
> │True │ False
> ▼     ▼
> 输出   跳过
> "你已成年"
>    │
>    ▼
> 程序结束
> ```

**常见错误**

> 1. **错误现象**:`IndentationError: expected an indented block`
>    **原因:**if 后面没缩进。修正:if 下一行缩进 4 格
> 2. **错误现象**:忘记冒号
>    **原因:**`if age >= 18` 没冒号会报错。修正:`if age >= 18:`

#### if-else 与 if-elif-else —— 多选一

> **痛点**:你需要在多个条件中选一个执行,比如成绩等级。
> **类比**:if-elif-else就像"多选题" —— 从上到下判断,第一个成立就执行,然后跳出。
> **解释**:`else` 是"以上都不成立时执行";`elif` 是"否则如果"。

```python
# if-else:二选一
age = 20
if age >= 18:
    print("成年")
else:
    print("未成年")

# if-elif-else:多选一(成绩等级)
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")      # 命中这里,跳出
elif score >= 60:
    print("及格")      # 不判断了(已命中)
else:
    print("不及格")
```

**逐行解剖**

- `if` = 第一个条件
- `elif` = 否则如果(可以有多个)
- `else` = 以上都不成立
- 从上到下判断,第一个成立就执行,后面不再判断

> **ASCII 多分支流程图**
> ```
> score = 85
>    │
>    ▼
> score >= 90 ? ──No──▶ score >= 80 ? ──Yes──▶ 输出"良好"
>    │                      │
>   Yes                    No
>    │                      ▼
> 输出"优秀"          score >= 60 ?
>                        │     │
>                       Yes    No
>                        ▼      ▼
>                      "及格"  "不及格"
> ```

**常见错误**

> 1. **错误现象**:多个条件都成立,但只执行第一个
>    **原因:**if-elif-else 是"多选一",命中一个就跳出
> 2. **错误现象**:elif 写成 else if
>    **原因:**Python 用 `elif` 不是 `else if`

#### 逻辑运算 —— 组合条件

> **痛点**:你想判断"年龄大于 18 且有身份证",一个条件不够。
> **类比**:逻辑运算就像"串联/并联" —— and 是串联(都通才通),or 是并联(一个通就通)。
> **解释**:`and` 两个都成立才 True;`or` 一个成立就 True;`not` 取反。

```python
# and:两个都成立
age = 20
has_id = True
if age >= 18 and has_id:
    print("可以进入")       # 两个条件都成立

# or:一个成立就行
is_vip = False
has_coupon = True
if is_vip or has_coupon:
    print("可以打折")       # 一个成立就执行

# not:取反
is_raining = False
if not is_raining:
    print("不用带伞")       # not False = True
```

**逐行解剖**

- `and` = 两个都为 True 才 True
- `or` = 至少一个为 True 就 True
- `not` = True 变 False,False 变 True

> **ASCII 逻辑运算真值表**
> ```
> and(与):          or(或):          not(非):
> T and T = T       T or T = T       not T = F
> T and F = F       T or F = T       not F = T
> F and T = F       F or T = T
> F and F = F       F or F = F
> ```

**常见错误**

> 1. **错误现象**:`if age >= 18 and <= 60:` 报错
>    **原因:**and 两边都要写完整条件。修正:`if age >= 18 and age <= 60:`
> 2. **错误现象**:混淆 `and` 和 `or`
>    **原因:**"且"用 and,"或"用 or。读题时先判断是"都"还是"任一"

#### 嵌套条件与三元表达式 —— 简写技巧

> **痛点**:条件里还要判断条件,代码越来越复杂;简单的 if-else 想写成一行。
> **类比**:嵌套条件就像"俄罗斯套娃" —— 大条件里套小条件;三元表达式就像"缩写"。
> **解释**:嵌套不超过两层;三元表达式 `值1 if 条件 else 值2` 是简写。

```python
# 嵌套条件(不超过两层)
age = 20
has_ticket = True
if age >= 18:
    if has_ticket:
        print("可以入场")
    else:
        print("需要购票")
else:
    print("未成年禁止入场")

# 三元表达式(简写)
age = 20
status = "成年" if age >= 18 else "未成年"
print(status)           # 成年

# in 运算符(判断是否在序列中)
fruit = "apple"
if fruit in ["apple", "banana", "orange"]:
    print("是水果")     # 成立
```

**逐行解剖**

- 嵌套 = if 里面再写 if(注意缩进叠加)
- 三元表达式 = `值1 if 条件 else 值2`,条件成立取值1,否则取值2
- `in` = 判断元素是否在列表/字符串中

> **ASCII 三元表达式示意图**
> ```
> status = "成年" if age >= 18 else "未成年"
>
> age >= 18 ?
>    │
> ┌──┴──┐
> │True │ False
> ▼     ▼
> "成年"  "未成年"
>    │
>    ▼
> status = "成年"
> ```

**常见错误**

> 1. **错误现象**:嵌套超过两层,代码难读
>    **原因:**嵌套太深可读性差。修正:用 and 合并条件或拆成函数
> 2. **错误现象**:三元表达式忘记 else
>    **原因:**三元表达式必须有 else。修正:`x if cond else y`

#### 综合实战 —— BMI 计算器

> **痛点**:学了条件分支,不知道怎么用在实际项目里。
> **类比**:BMI 计算器就像"体检报告" —— 输入身高体重,程序判断你胖不胖。
> **解释**:结合输入、计算、条件判断,做一个实用小工具。

```python
# BMI 计算器
# BMI = 体重(kg) / 身高(m)^2
# 偏瘦:<18.5  正常:18.5-24  偏胖:25-28  肥胖:>28

weight = float(input("体重(kg):"))
height = float(input("身高(m):"))
bmi = weight / (height ** 2)

print(f"BMI = {bmi:.1f}")

if bmi < 18.5:
    print("偏瘦")
elif bmi < 24:
    print("正常")
elif bmi < 28:
    print("偏胖")
else:
    print("肥胖")
```

**逐行解剖**

- `float(input(...))` = 输入并转浮点数
- `bmi = weight / (height ** 2)` = 计算 BMI
- `if-elif-else` = 多选一判断等级
- `:.1f` = 保留 1 位小数
