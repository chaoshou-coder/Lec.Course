# Python 零基础 → AI 就绪工程师

> 从零开始,循序渐进,用 Python 打开 AI/数据科学的大门。

---

## 如何使用本仓库

1. **从这里开始**:按下面的知识点顺序学习,每个知识点是一个独立 Markdown 文件
2. **阅读知识点**:打开 `knowledge/` 下的 md 文件,理解概念和示例
3. **动手练习**:md 文件中标注了"打开 `exercises/xxx/practice01.py`",用 VS Code 打开并补全代码
4. **对照答案**:在 `solutions/xxx/` 下找到对应参考答案
5. **跳转导航**:每个知识点末尾有"下一个/上一个"链接,点击跳转

> **不需要按天规划**:按你自己的节奏,一天学 1-3 个知识点都可以。

## 学习建议

- **每天 1-3 个知识点**,量力而行
- **先读后练**:先通读 md 概念,再打开 exercises 练习
- **不要跳练习**:每个练习都做,才能真正掌握
- **遇到困惑**:每个知识点都有"常见错误"段,先自查
