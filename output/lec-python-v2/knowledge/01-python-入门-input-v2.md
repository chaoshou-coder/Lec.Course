#### input() 函数 —— 读取用户输入

> **痛点**

之前的程序只能处理**写死**在代码里的数据,
每次修改数据都要改源代码,再重新运行。
能不能让程序**运行时**才问用户要数据?

> **类比**

`input()` 就像考试卷上的**填空题**:
卷子上印着"请输入你的名字:____",
考生填什么,答案就是什么。
程序出题(`input("提示")`),用户作答(键盘输入)。

> **解释**

`input("提示文字")` 会在屏幕上显示提示文字,
**暂停**程序,等用户从键盘输入内容并按回车,
然后把用户输入的内容以**字符串**形式返回。

**ASCII 内存图 —— input() 的执行**

```
没有 input():
  程序 ──→ 自己算 ──→ 输出结果(数据写死在代码里)

有 input():
  程序 ──→ 显示 "请输入年龄:" ──→ ⏸️ 暂停等待
                                │
                          用户输入 "18" 回车
                                │
                                ▼
  age = "18" ←── input() 返回字符串 "18"
  程序继续执行

内存:
  用户输入前:  age 不存在
  用户输入后:  age = "18"   ← 注意:是字符串 "18",不是数字 18
```

**关键**:`input()` **永远返回字符串**,
即使输入的是数字,拿到的也是字符串 `"18"` 而不是整数 `18`。

```python
# 示例 1:input() 的基本用法
# --- 执行过程 ---
# 第 1 行 name = input("请输入你的名字:"):
#   ① 在屏幕显示 "请输入你的名字:",光标等待
#   ② 用户从键盘输入,比如 "张三",按回车
#   ③ input() 把用户输入的内容作为字符串返回 → "张三"
#   ④ 创建变量 name,指向字符串 "张三"
#
# 第 2 行 print(name):
#   ① 读取 name → "张三"
#   ② 输出: 张三
#
# 第 3 行 print(type(name)):
#   ① type(name) 判断 name 的类型 → <class 'str'>
#   ② 输出: <class 'str'>

name = input("请输入你的名字:")   # 提示用户输入名字,返回字符串
print(name)                       # 输出用户输入的内容
print(type(name))                 # 输出类型 → <class 'str'>
```

**逐行解剖**

① `input("请输入你的名字:")` 显示提示,等待输入,返回字符串
② `name = ...` 把 input() 返回的字符串赋给变量 name
③ `print(name)` 把 name 的值打印到屏幕
④ `print(type(name))` 验证类型 —— 永远是 `str`(字符串)

**要点**:`input()` 的返回值**永远是字符串**,
不管用户输入的是 `123` 还是 `3.14`,
拿到的都是字符串 `"123"` `"3.14"`。

```python
# 示例 2:input() 配合类型转换 —— 年龄加 5 岁
# --- 执行过程 ---
# 第 1 行 age_str = input("请输入年龄:"):
#   ① 显示 "请输入年龄:",等待用户输入
#   ② 用户输入 "18",回车
#   ③ input() 返回字符串 "18"
#   ④ 创建变量 age_str,指向字符串 "18"
#
# 第 2 行 age = int(age_str):
#   ① int(age_str) 把字符串 "18" 转成整数 18
#   ② 创建变量 age,指向整数 18
#
# 第 3 行 future = age + 5:
#   ① 读取 age → 18
#   ② 18 + 5 = 23
#   ③ 创建变量 future,指向整数 23
#
# 第 4 行 print(future):
#   ① 读取 future → 23
#   ② 输出: 23

age_str = input("请输入年龄:")   # 输入 "18",返回字符串 "18"
age = int(age_str)               # 把字符串 "18" 转成整数 18
future = age + 5                 # 18 + 5 = 23
print(future)                    # 输出: 23
```

**逐行解剖**

① `age_str = input("请输入年龄:")` 拿到的是字符串 `"18"`
② `age = int(age_str)` 用 `int()` 把字符串转成整数 `18`
③ `future = age + 5` 整数才能做数学运算 → `23`
④ `print(future)` 输出结果

**要点**:`input()` 拿到字符串,要做数学运算必须**先转类型**。

```python
# 示例 3:类型转换的简写 —— 一步到位
# --- 执行过程 ---
# 第 1 行 price = float(input("请输入价格:")):
#   ① 先执行 input("请输入价格:") → 显示提示,等待输入
#   ② 用户输入 "59.8",回车 → input() 返回字符串 "59.8"
#   ③ float("59.8") 把字符串转成浮点数 59.8
#   ④ 创建变量 price,指向浮点数 59.8
#
# 第 2 行 discounted = price * 0.8:
#   ① 读取 price → 59.8
#   ② 59.8 * 0.8 = 47.84
#   ③ 创建变量 discounted,指向浮点数 47.84
#
# 第 3 行 print(discounted):
#   ① 读取 discounted → 47.84
#   ② 输出: 47.84

price = float(input("请输入价格:"))  # input 返回 "59.8",float 转成 59.8
discounted = price * 0.8             # 59.8 * 0.8 = 47.84
print(discounted)                    # 输出: 47.84
```

**逐行解剖**

① `input(...)` 先执行,返回字符串 `"59.8"`
② `float(...)` 把字符串转成浮点数 `59.8`
③ `price = ...` 把浮点数赋给变量 price
④ `price * 0.8` 浮点数可以做乘法 → `47.84`
⑤ `print(discounted)` 输出打折后的价格

**要点**:`float(input(...))` 是常见写法,
`input()` 返回字符串,`float()` 转成小数。

**NCDL Break It 演示(input() 反模式)**

```python
# ============ BREAK IT 演示 ============
# 错误 1:直接把 input() 返回值当数字用
try:
    age = input("请输入年龄:")    # 输入 "18",age = "18"(字符串)
    result = age + 5             # ❌ 字符串 + 整数!
except TypeError as e:
    print(f"报错: {e}")
    # 输出: 报错: can only concatenate str (not "int") to str

# 错误 2:输入字母却想转成整数
try:
    num = int(input("请输入整数:"))  # 用户输入 "abc"
except ValueError as e:
    print(f"报错: {e}")
    # 输出: 报错: invalid literal for int() with base 10: 'abc'
# ============ END BREAK IT ============
```

**常见错误**

> 1. **错误现象**:`TypeError: can only concatenate str (not "int") to str`
>    **原因:**`input()` 返回字符串,字符串不能和整数做加法。修正:`age = int(input("请输入年龄:")),再做加法`
> 2. **错误现象**:`ValueError: invalid literal for int() with base 10: 'abc'`
>    **原因:**用户输入了非数字字符串 `"abc"`,`int()` 无法转换。修正:确保输入的是数字,或先判断再转换
> 3. **错误现象**:忘记写 `input()` 的提示参数,屏幕没有任何提示,用户不知道要输入什么
>    **原因:**`input()` 可以不加参数 `input()`,但用户体验差。修正:`input("请输入年龄:")` 加上提示文字
