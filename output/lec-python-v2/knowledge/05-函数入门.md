#### 为什么需要函数 —— DRY 原则

> **痛点**

代码里出现 5 段长得一样的代码,
改 bug 要改 5 处。复制粘贴越多,越痛。

> **类比**

函数 = 咖啡机:放进去原料(参数),
按按钮(调用),出来咖啡(返回值)。
不用每次从头磨豆子。

> **解释**

函数把重复逻辑**打包成一个名字**,
定义一次,调用无数次。
核心原则:DRY = Dont Repeat Yourself。

```python
# 反例:复制粘贴
s = "  hello  "
s = s.strip()
s = s.upper()
print(s)

s = "  world  "
s = s.strip()
s = s.upper()
print(s)

s = "  python  "
s = s.strip()
s = s.upper()
print(s)

# 改进:把重复逻辑打包成函数,改一处就行
def format(s):
    s = s.strip()
    s = s.upper()
    return s

print(format("  hello  "))
print(format("  world  "))
print(format("  python  "))
```

**常见错误**

> 1. **觉得复制粘贴方便**:短期方便,长期维护难
> 2. **函数写太长**:一个函数应该只做一件事

> **问自己**:
> - 如果需求变了(要去掉两端*和空白),要改几处?
> - 如果逻辑更复杂(10 步),改一处 vs 改十处?
> - 函数能解决所有重复问题吗?

#### def 定义和调用 —— 给一段代码起个名字

> **解释**

`def 函数名(参数):` 定义函数,不会立刻执行。
`函数名(参数)` 触发调用。

```python
# 示例:定义和调用函数
def say_hi():
    print("Hello!")
    print("欢迎学习 Python")

say_hi()    # 第 1 次调用
say_hi()    # 第 2 次调用
say_hi()    # 第 3 次调用

# 逐行解剖
# ① def say_hi(): 定义函数,不立刻执行
# ② 函数体缩进 4 格
# ③ say_hi() 触发调用,跳到函数体执行
# ④ 调用几次,函数体执行几遍
```

**常见错误**

> 1. **忘调用**:写了 def 不等于"跑一次",必须调用
> 2. **忘冒号**:`def say_hi()` → SyntaxError
> 3. **缩进错**:函数体不缩进 → IndentationError

> **问自己**:
> - 如果写了 def 但不调用,里面的代码执行吗?
> - 调用 3 次,函数体执行几遍?
> - 函数名能不能和变量名一样?

#### 函数四种形式

| 形式 | 参数 | 返回值 | 使用频率 |
| --- | --- | --- | --- |
| 无参无返回 | 无 | 无 | 中(打印菜单) |
| 有参无返回 | 有 | 无 | 中(打印定制文案) |
| 无参有返回 | 无 | 有 | 低 |
| **有参有返回** | 有 | 有 | **最高** |

> **解释**

第 4 种(有参有返回)最常用!其他都是它的特例。

```python
# 示例:四种形式的函数
def menu():
    """无参无返回:打印菜单"""
    print("1. 添加  2. 删除  3. 退出")

def greet(name):
    """有参无返回:打印定制文案"""
    print(f"你好,{name}")

def get_answer():
    """无参有返回:返回固定值"""
    return 42

def add(a, b):
    """有参有返回:返回计算结果"""
    return a + b

# 调用
menu()
greet("小红")
print(get_answer())
print(add(2, 3))
```

#### 形参/实参/默认参数/关键字参数

> **痛点**

- 形参是"坑位",实参是"值"
- 默认参数让某些参数"可选填"
- 关键字参数让顺序无关

```python
# 示例:形参/实参/默认参数/关键字参数
def introduce(name, age, city):
    print(f"{name},{age} 岁,来自 {city}")

# 位置传参(按位置)
introduce("小明", 18, "北京")

# 关键字参数(按名字,顺序无关)
introduce(age=20, city="上海", name="小红")

# 默认参数
def greet(name="学员"):
    print(f"你好,{name}")

greet()          # 使用默认值
greet("张三")    # 覆盖默认值

# 逐行解剖
# ① introduce("小明", 18, "北京") 按位置赋值
# ② introduce(age=20, ...) 按名字赋值,顺序无关
# ③ greet() 没传参,使用默认值 "学员"
```

**常见错误**

> 1. **默认参数放前面**:`def f(a=1, b)` → SyntaxError
>    **原因:**默认参数必须在非默认参数后面
> 2. **位置参数在关键字参数后**:`f(a=1, 2)` → SyntaxError

> **问自己**:
> - `def f(a, b=1)` 和 `def f(a=1, b)` 哪个合法?
> - 关键字参数有什么用?
> - 如果定义 `def f(a, b=1)` 可以只传 a 吗?

#### return —— 带结果回家 + 立刻退出

> **解释**

`return` 把结果送回调用处,
同时立刻退出函数(后面代码不执行)。

```python
# 示例:return 的两种用法
def check_age(age):
    if age < 0:
        return "年龄不能为负"
    if age < 18:
        return "未成年"
    return "成年"

print(check_age(20))    # 成年
print(check_age(15))    # 未成年
print(check_age(-5))    # 年龄不能为负

# 逐行解剖
# ① check_age(20):20<0? 否 → 20<18? 否 → return "成年"
# ② check_age(15):15<0? 否 → 15<18? 是 → return "未成年"
# ③ check_age(-5):-5<0? 是 → return "年龄不能为负"
```

> **BREAK IT:return 后写代码**

```python
# ============ BREAK IT 演示 ============
# def broken():
#     return "结果"
#     print("这行永远不会执行!")  # 死代码
#
# print(broken())  # 只输出 "结果"
# ============ END BREAK IT ============
```

**常见错误**

> 1. **以为 return 后还执行**:return 立刻退出,后面是死代码
> 2. **return 无值**:return None,函数无返回值
> 3. **忘记 print 返回值**:需要 `print(f(x))` 才能看到结果

> **问自己**:
> - return 后的代码会执行吗?
> - return 和 print 有什么区别?
> - 一个函数能写多个 return 吗?

#### 多值返回 —— 元组解包

> **解释**

`return a, b, c` 本质返回元组,
用逗号变量接收叫"解包"。

```python
# 示例:多值返回 + 元组解包
def calculate(a, b):
    return a + b, a - b, a * b

sum_, diff, prod = calculate(10, 3)
print(sum_, diff, prod)    # 13 7 30

# 逐行解剖
# ① calculate(10, 3) 返回 (13, 7, 30) 元组
# ② 元组解包:sum_=13, diff=7, prod=30
# ③ 本质:return 13, 7, 30 == return (13, 7, 30)
```

**常见错误**

> 1. **变量数量不匹配**:`a, b = f()` 但 f 返回 3 个值 → ValueError
> 2. **以为返回多个独立值**:实际返回一个元组

> **问自己**:
> - 多值返回的本质是什么?
> - 如果只写一个变量接收,变量类型是什么?
> - 元组解包还能用在哪些地方?

#### 作用域 —— 全局 vs 局部

> **解释**

- 全局变量:函数外定义,函数内可读
- 局部变量:函数内定义,函数外不可见
- 函数内不能直接修改全局变量(除非 global)

```python
# 示例:作用域的基本规则
x = 10          # 全局变量

def foo():
    y = 20      # 局部变量
    print(x)    # 函数内可读全局变量 → 10
    print(y)    # 函数内可读局部变量 → 20

foo()
# print(y)      # 报错!y 是局部变量,函数外不可见

# 逐行解剖
# ① x = 10 创建全局变量
# ② y = 20 创建局部变量(只在函数内有效)
# ③ print(x) 函数内没有局部 x,沿局部→全局查找
# ④ 函数结束,局部变量 y 被销毁
```

> **BREAK IT:函数内改全局变量**

```python
# ============ BREAK IT 演示 ============
# x = 10
# def broken():
#     x = x + 1   #  UnboundLocalError!
#     print(x)
#
# broken()
# ============ END BREAK IT ============

# 原因:函数内对 x 赋值,Python 认为 x 是局部变量
# 但局部 x 还没定义就 x+1,报错
# 修正:用 global x 声明
```

**常见错误**

> 1. **函数内改全局变量**:`x += 1` 报 UnboundLocalError
>    **原因:**Python 认为 x 是局部变量。修正:`global x`
> 2. **以为局部变量函数外能用**:函数结束局部变量销毁

> **问自己**:
> - 函数内能读全局变量吗?
> - 函数内能改全局变量吗?
> - 函数外能读局部变量吗?

#### 综合练习 —— 工具函数库

定义 5 个工具函数:
1. `is_valid_phone(phone)` → 返回 bool(11 位且全数字)
2. `mask_phone(phone)` → 返回 "138****8000"
3. `bmi_report(h, w)` → 返回评级字符串
4. `is_leap(year)` → 返回 bool
5. `grade(score)` → 返回 "A/B/C/D/F"

```python
# 示例:工具函数库完整实现
def is_valid_phone(phone):
    return len(phone) == 11 and phone.isdigit()

def mask_phone(phone):
    return phone[:3] + "****" + phone[-4:]

def bmi_report(h, w):
    bmi = w / (h ** 2)
    if bmi < 18.5:
        return "偏瘦"
    if bmi < 24:
        return "正常"
    if bmi < 28:
        return "偏胖"
    return "肥胖"

def is_leap(year):
    return (year % 4 == 0 and year % 100 != 0) or year % 400 == 0

def grade(score):
    if score >= 90:
        return "A"
    if score >= 80:
        return "B"
    if score >= 70:
        return "C"
    if score >= 60:
        return "D"
    return "F"

# 测试
print(is_valid_phone("13812345678"))  # True
print(mask_phone("13812345678"))      # 138****5678
print(bmi_report(1.75, 70))           # 正常
print(is_leap(2024))                  # True
print(grade(85))                      # B
```
